# Screenshot Slots

Place portfolio screenshots in this folder using these filenames:

- `dashboard.png`
- `calendar.png`
- `dialog.png`
- `list.png`
- `completed.png`
- `mobile.png`

Suggested capture order:

1. Hero dashboard with stats and tab pills
2. Calendar month view with marked deadlines
3. Add or edit task dialog
4. Active list with styled task cards
5. Completed tab
6. Mobile viewport framing

Tips:

- Keep device chrome consistent across all images.
- Use light mode for this project, since the UI palette was designed around warm surfaces.
- Prefer clean sample data with varied deadlines so the calendar markers read clearly.
